a > display: block; 不给宽度, a 自动适应固定的宽度的父元素的宽度，不管添加多大的 margin,padding 

      .box1 {
        border: 1px solid blue;
        width: 300px;
        margin-bottom: 10px;
      }
      .box1 > a {
        /* 不设置 width*/
        display: block;
        border: 1px solid red;
        margin: 0 10px;
        padding: 0 30px;
      }
a1 a2 a3 a4

a > display: block; width: auto; a 自动适应固定的宽度的父元素的宽度，不管添加多大的 margin,padding 

      .box2 {
        border: 1px solid blue;
        width: 300px;
      }
      .box2 > a {
        /* 设置 width: auto;*/
        display: block;
        width: auto;
        border: 1px solid red;
        margin: 0 10px;
        padding: 0 30px;
      }
a1 a2 a3 a4

a > display: block; width: 100%; 添加 margin,padding，a 会超出固定的宽度的父元素 

      .box3 {
        border: 1px solid blue;
        width: 300px;
        margin-top: 10px;
      }
      .box3 > a {
        /* 设置 width: 100%; 这样添加的padding,margin会撑开父盒子，父盒子固定宽度*/
        display: block;
        width: 100%;
        border: 1px solid red;
        margin: 0 10px;
        padding: 0 30px;
      }
a1 a2 a3 a4
