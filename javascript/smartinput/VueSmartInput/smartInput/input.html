智能输入框

类似select标签，但能够支持输入框的过滤和多选功能

代码演示

基本用法

简单的输入框，支持数据的列举，支持鼠标点选和键盘控制。

代码
  • 天津市
当前值：天津市 

<template>
    <div id="demo">
        <smart-input v-bind="provinceList" @collect="collectProvince"></smart-input>
    </div>
</template>

<script>
const provinceList = ['北京市','天津市','上海市','重庆市','河北省','山西省','辽宁省','吉林省','黑龙江省','江苏省','浙江省','安徽省','福建省','江西省','山东省','河南省','湖北省','湖南省','广东省','海南省','四川省','贵州省','云南省','陕西省','甘肃省','青海省','台湾省','内蒙古自治区','广西壮族自治区','西藏自治区','宁夏回族自治区','新疆维吾尔自治区','香港特别行政区','澳门特别行政区'];
const vueDemo = new Vue({
    el: '#demo',
    data: {
        provinceList: {
            list: provinceList,
        }
    },
    methods: {
        // 跟智能输入框同步选中的省份
        collectProvince(data) {
            console.log(data);
        }
    }
});
</script>

支持多选

通过添加multiple: true配置来支持多选。

代码
  • 上海市
当前值：[ "上海市" ] 
<template>
    <div id="demo">
        <smart-input v-bind="provinceList" @collect="collectProvince"></smart-input>
    </div>
</template>

<script>
const provinceList = ['北京市','天津市','上海市','重庆市','河北省','山西省','辽宁省','吉林省','黑龙江省','江苏省','浙江省','安徽省','福建省','江西省','山东省','河南省','湖北省','湖南省','广东省','海南省','四川省','贵州省','云南省','陕西省','甘肃省','青海省','台湾省','内蒙古自治区','广西壮族自治区','西藏自治区','宁夏回族自治区','新疆维吾尔自治区','香港特别行政区','澳门特别行政区'];

    data: {

        }

    methods: {
        // 跟智能输入框同步选中的省份

            console.log(data);

支持初始值

通过传入value: ['河南省', '错误值']来支持初始值，会对非法字段进行过滤。

代码 
<template>
    <div id="demo">
        <smart-input v-bind="provinceList" @collect="collectProvince"></smart-input>
    </div>
</template>

<script>
const provinceList = ['北京市','天津市','上海市','重庆市','河北省','山西省','辽宁省','吉林省','黑龙江省','江苏省','浙江省','安徽省','福建省','江西省','山东省','河南省','湖北省','湖南省','广东省','海南省','四川省','贵州省','云南省','陕西省','甘肃省','青海省','台湾省','内蒙古自治区','广西壮族自治区','西藏自治区','宁夏回族自治区','新疆维吾尔自治区','香港特别行政区','澳门特别行政区'];
const vueDemo = new Vue({
    el: '#demo',
    data: {
        provinceList: {
            list: provinceList,
            // 支持数据初始化
            multiple: true,
            value: ['河南省', '错误值']
        }
    },
    methods: {
        // 跟智能输入框同步选中的省份
        collectProvince(data) {
            console.log(data);
        }
    }
});
</script>
